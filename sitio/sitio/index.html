01 — ACERCA DEL PROYECTO

Flujo de trabajo DevOps completo

Este sitio fue creado como parte de la Actividad 1 del curso de Desarrollo DevOps en el ITESM. El objetivo fue implementar un flujo donde Git funciona como fuente única de verdad y cada cambio versionado se refleja automáticamente en producción mediante AWS S3.

Automatización

Script PowerShell que despliega con un solo comando desde la terminal.
